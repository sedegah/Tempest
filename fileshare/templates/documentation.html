Overview

Introduction

Tempest is an open-source, ephemeral file-sharing platform designed for maximum privacy and minimal footprint. Built on Django and Cloudflare R2, it provides a "Snapchat for files" experience with military-grade encryption.

Getting Started

To deploy your own instance of Tempest, you'll need a Cloudflare account with R2 enabled and a Python environment.

git clone https://github.com/sedegah/tempest
pip install -r requirements.txt
python manage.py migrate

Encryption Layer

We use the cryptography library's Fernet implementation (AES-256 in CBC mode with HMAC-SHA256). Each file is encrypted with a unique key that never touches our long-term logs.
