Frequently Asked Questions

Everything you need to know about Tempest.

When a file is downloaded, the SharedFile model sets downloaded=True. The is_expired() method returns True, and all subsequent requests return a 410. The link is permanently invalid after first use.
Encryption is optional and uses the Python cryptography library with AES-256 (Fernet). A unique key is generated per file, stored in the DB, and used to decrypt on-the-fly when the file is served.
